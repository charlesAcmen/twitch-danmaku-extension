# Privacy Policy / 隐私政策

**Last Updated / 最后更新:** 2026-06-13

## English

### Data Collection
This extension **does NOT collect, store, or transmit any personal data or user information**.

### Storage Permission
The extension requests the `storage` permission solely for:
- Saving your danmaku display preferences (opacity, font size, speed, etc.)
- All settings are stored locally in your browser using `localStorage`
- No data is ever sent to external servers or third parties

### Network Activity
This extension operates entirely client-side and:
- Does NOT make any external API calls
- Does NOT communicate with any third-party services
- Does NOT track your browsing activity
- Does NOT use analytics or telemetry

### Third-Party Emotes
The extension passively scans visible Twitch chat DOM elements to extract emote information that has already been loaded by your browser. This process:
- Does NOT make API calls to 7TV, BTTV, FFZ, or other third-party services
- Does NOT access or store authentication tokens
- Only reads publicly visible DOM elements in your browser
- Is completely ban-safe and respects Twitch's Terms of Service

### Open Source
This extension is open source. You can review the complete source code to verify these privacy claims.

## 中文

### 数据收集
本扩展**不收集、存储或传输任何个人数据或用户信息**。

### 存储权限
扩展请求 `storage` 权限仅用于：
- 保存您的弹幕显示偏好设置（不透明度、字体大小、速度等）
- 所有设置使用 `localStorage` 本地存储在您的浏览器中
- 数据永远不会发送到外部服务器或第三方

### 网络活动
本扩展完全在客户端运行，并且：
- 不进行任何外部 API 调用
- 不与任何第三方服务通信
- 不跟踪您的浏览活动
- 不使用分析或遥测

### 第三方表情
扩展被动扫描可见的 Twitch 聊天 DOM 元素，以提取浏览器已加载的表情信息。此过程：
- 不对 7TV、BTTV、FFZ 或其他第三方服务进行 API 调用
- 不访问或存储身份验证令牌
- 仅读取浏览器中公开可见的 DOM 元素
- 完全防封号，尊重 Twitch 的服务条款

### 开源
本扩展是开源的。您可以查看完整的源代码以验证这些隐私声明。
